# BloodPressureTracker — Дневник давления

Мобильное приложение для Android на Kotlin. Позволяет вести дневник артериального давления, отслеживать динамику на графике, устанавливать напоминания.

## Функциональность

- Добавление, редактирование, удаление записей давления (систолическое, диастолическое, пульс, дата/время).
- Автоматическая интерпретация уровня давления (норма, гипертония, криз и т.д.).
- Просмотр списка записей в хронологическом порядке.
- График изменения систолического и диастолического давления.
- Ежедневные напоминания о необходимости измерить давление (WorkManager + уведомления).
- Боковое меню для навигации.

## Технологии

- **Kotlin** – язык разработки.
- **Android SDK** (minSdk 24, targetSdk 34).
- **Room** – локальная база данных.
- **LiveData + ViewModel** – реактивное хранение данных.
- **RecyclerView** – отображение списка записей.
- **MPAndroidChart** – построение графиков.
- **WorkManager** – планирование напоминаний.
- **Material Design** – интерфейс.
